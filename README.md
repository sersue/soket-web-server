## soket-web-server
2020-1 Computer Network Project

## 프로젝트의 목표
• C언어를 이용하여 웹서버를 구현한다. (타 언어 허용 불가)  
• 서버와 웹브라우저의 동작 방식에 대해 학습한다.  

##  프로젝트 내용  
• Part 1: Client의 request message를 화면에 출력하는 "Web server" 제작  
• Part II: Web server가 browser의 request에 response할 수 있도록 확장
